# go-mdwiki

## Basic Golang Markdown Wiki

- super simple :)
- one binary + config file
- inspired by go-bwiki
- no user accounts
- pages are stored as plain markdown files 

## How to run

```
go run .
```

## Configuration

```
host: localhost
port: 8080
storage: .storage
```

- host & post - web server properties
- storage is directory where pages going to be stored
